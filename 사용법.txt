염색도우미 사용법 안내
-----------------------

[1] 색상 입력
- 좌/중앙/우 영역별로 색상을 선택합니다.
- RGB 또는 HEX를 입력할 수 있으며, 서로 자동으로 변환됩니다.
- 색상 미리보기를 통해 입력한 색상을 즉시 확인할 수 있습니다.
- 각 색상의 "반드시 포함"을 체크하면 해당 색상은 반드시 결과에 포함되어야 합니다.

[2] 허용 오차 설정
- 색상 비교에 사용할 오차 범위를 설정하세요.
- 오차가 작을수록 더 정확한 색상만 일치로 판단합니다.
- "반드시 포함"된 색상은 오차 없이 정확히 일치해야 합니다.

[3] 팔레트 영역 지정
- "팔레트 영역 지정하기" 버튼을 눌러 화면의 팔레트를 드래그로 지정합니다.
- 지정된 영역은 이후 자동 인식 및 조작에 사용됩니다.

[4] 자동 인식 시작
- "자동 인식 시작" 버튼을 누르면 흰 원을 감지하고, 원하는 색상과 비교하여 자동으로 조작합니다.
- 자동화 도중 조건에 맞는 색상이 감지되면 자동화가 종료됩니다.
- 중지하려면 "정지" 버튼 또는 단축키 [Esc]를 누르세요.

[5] 단축키
- 자동 시작: Ctrl + Enter
- 자동 중지: Esc

-----------------------

📌 참고
- RGB/HEX → 상호 입력 가능하며 자동 변환됩니다.
- 색상 분석 결과로 색상(H), 채도(S), 명도(V)를 UI에 자동 표시합니다.
- 흰 원이 감지되지 않으면 계속 반복 시도합니다.
